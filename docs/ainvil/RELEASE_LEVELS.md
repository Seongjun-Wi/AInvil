# AInvil Release Levels

[Back to README](../../README.md)

## Current Level

AInvil is currently:

```text
Core Release Ready / Release Candidate
Product MVP Ready Candidate
Public Release Ready: No
```

## Definitions

| Level | Meaning | Current Status |
| --- | --- | --- |
| Core Release Ready | Core plugin, Unity Bridge smoke, evidence pipeline, review, and release reports are verified. | Yes |
| Core RC Reproducibility Verified | The release candidate can be checked through regression and report validators. | Yes |
| Product MVP Ready Candidate | A single Unity project has a playable Product MVP vertical slice with evidence. | Yes |
| Public Release Ready | Public installation, onboarding, broad environment handling, and release polish are ready. | No |

## Why Public Release Ready Is Not Claimed

Remaining gaps:

- public installer
- polished user-facing setup
- broader multi-project benchmark
- long-session stability
- stronger onboarding UX
- bridge watchdog / auto-recovery
- production-level art, sound, tutorial, save/load, and progression loops

## Safe Language

Use:

```text
Core Release Ready / Release Candidate
Product MVP Ready Candidate
Evidence-grounded Unity workflow
Single-project case study
```

Avoid:

```text
Public Release Ready
Production-finished commercial game
Verified across all Unity projects
Fully automatic game production
Human review unnecessary
```
