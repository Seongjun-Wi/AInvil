# AInvil 한국어 문서

이 폴더는 AInvil을 "작은 아이디어라도 실현시켜주는 AI 게임 회사/플랫폼"으로 출시하기 위한 한국어 기획 문서 모음이다.

## 문서 목록

| 문서 | 용도 |
| --- | --- |
| `AInvil_제품_출시_기획서.md` | 출시까지 부족한 부분, 보완 방향, 제품 기능, MVP 범위, 릴리스 게이트를 정리한다. |
| `AInvil_기술_기획서.md` | 제품 기능을 실제로 구현하기 위한 아키텍처, 모듈, 데이터, 검증, 로드맵을 정리한다. |

## 권장 읽기 순서

## 현재 결론

AInvil은 이미 기획, Unity 연동, 검증 철학, 상태 관리의 기반을 갖고 있다. 출시 전 핵심 과제는 공식 sample vertical slice를 반복 성공시키고, 온보딩/대시보드/검증 증거/문서 동기화를 하나의 제품 경험으로 묶는 것이다.
